# 🎉 Welcome to the Stay Tooned GFX Lesson Series

> *You're about to build a real, live webcomic website from scratch —
> and it's going to have all the good stuff: animated backgrounds,
> pixel-dissolve page transitions, custom animated cursors, and a
> first-visit welcome modal. No WordPress. No frameworks. Just you,
> some HTML/CSS/JS, and GitHub.*

---

## 🗺️ What You're Building

Think of a webcomic website like **a comic book store you own.**

- The **building itself** is your folder structure — where every file lives
- The **display cases** are your HTML pages — the bones of what the reader sees
- The **paint and signs** are your CSS — what everything looks like
- The **staff who move things around** are your JavaScript files — the behaviour
- The **inventory system** is `scan.js` — it reads your comics folder and writes a checklist
- The **back room** is GitHub — where the real copy lives, published to the world

By the end of these lessons you'll have:

✅ A multi-series comic site hosted free on GitHub Pages  
✅ A tiled animated TV-static wallpaper that visitors can control  
✅ Pixel-dissolve page transitions between every page  
✅ Custom animated cursors that change per-series  
✅ A first-visit welcome overlay with an email sign-up  
✅ A full comic reader with zoom, swipe, keyboard shortcuts  
✅ Automatic content detection — drop images in a folder, run one command

## 🎒 What You Need Before Lesson 1

### Software (free unless noted)

| Tool | Why | Where |
|------|-----|-------|
| **VS Code** | Code editor | [code.visualstudio.com](https://code.visualstudio.com) |
| **Node.js** (LTS) | Runs the scanner script | [nodejs.org](https://nodejs.org) |
| **Git** | Version control + deploy | [git-scm.com](https://git-scm.com) |
| **Chrome** or **Firefox** | DevTools are essential | Already installed |
| **Clip Studio Paint** or similar | Drawing your comics | Paid |

### Accounts (free)

| Service | Why | Where |
|---------|-----|-------|
| **GitHub** | Hosts your site for free | [github.com](https://github.com) |
| **Mailchimp** (optional) | Email newsletter | [mailchimp.com](https://mailchimp.com) |

### Skills assumed

- You know what HTML tags are (`<div>`, `<img>`, `<a>`, etc.)
- You've written a little CSS before
- You can open a terminal / command prompt and type commands
- You don't need to know JavaScript — we'll walk through every piece

> 💡 **Never used a terminal before?** On Windows: press `Win+R`, type `cmd`, press Enter.
> On Mac: press `Cmd+Space`, type `terminal`, press Enter.
